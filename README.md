# Focus-Test
Keeps track of Windows apps that have focus.

## How to use
1. Launch FocusTest
2. Work as usual
3. FocusTest will list all apps that have taken focus since it launched  
![Screenshot of FocusTest][1]
4. Compare PID with Task Manager (Ctrl + Shift + Esc)
